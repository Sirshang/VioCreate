# VioCreate - 莱米多媒体创作工具（前端项目）

纯前端项目，包含主页、项目页、设置弹窗与基础按钮跳转，不实现后端功能。

## 项目结构

```
VioCreate/
├── index.html      # 入口页面（单页应用）
├── css/
│   └── main.css    # 全局样式
├── js/
│   └── app.js      # 路由、弹窗、Toast、按钮事件
└── README.md
```

## 运行方式

### 方式一：直接打开（推荐）

双击 `index.html` 或在浏览器中打开该文件。

### 方式二：本地静态服务（避免部分浏览器对 file:// 的跨域限制）

```bash
# 若已安装 Node.js，可使用 npx 启动简单静态服务
npx serve .

# 或使用 Python
python3 -m http.server 8080
# 然后访问 http://localhost:8080
```

## 功能说明（仅前端交互）

| 操作 | 效果 |
|------|------|
| 顶部 **设置** | 打开应用设置弹窗 |
| 主页 **新建项目** | 打开创建项目弹窗 |
| 主页 **打开项目** | 跳转到项目工作区（URL 变为 `#project`） |
| 项目页 **返回主界面** | 返回项目管理页（URL 变为 `#`） |
| **刷新** / **批量处理** | 底部 Toast 提示（演示） |
| **文件 / 客服 / 帮助** | Toast 提示（演示） |
| 设置内 **大模型 / 图像生成 / 视频生成 / 翻译 / 剪映** | 标签页切换 |
| 设置 **保存并应用** | 关闭弹窗并 Toast「设置已保存」 |
| 创建项目 **开始处理** | 关闭弹窗并 Toast「项目已创建」 |
| **重置为默认值** | Toast 提示 |

## 技术栈

- HTML5
- [Tailwind CSS](https://tailwindcss.com/)（CDN）
- [Iconify](https://iconify.design/)（CDN）
- 原生 JavaScript（无框架）

## 说明

- 本仓库仅做界面与前端交互演示，不包含任何后端接口或真实业务逻辑。
- 如需对接后端，可在 `js/app.js` 中在对应 `data-action` 回调里增加请求逻辑。
